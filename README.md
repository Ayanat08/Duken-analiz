Техникалық тапсырма (ТЗ)

Жоба атауы: SmartSales – Тұтынушылардың мінез-құлқын талдау және сұранысты болжау жүйесі.

Жобаның мақсаты: Дүкендегі сатып алушылардың бұрынғы транзакцияларын талдай отырып, болашақтағы тауарға деген сұранысты болжайтын заманауи веб-платформа құру. Жүйе бизнес иелеріне қойманы тиімді басқаруға, сауданы арттыруға және артық тауар шығындарын болдырмауға көмектеседі. Жоба қай тауардың тез өтетінін анықтауға, клиенттердің қызығушылығына қарай жеке ұсыныстар дайындауға және тауар айналымын бақылауға бағытталған.

Мақсатты аудитория: Шағын және орта бизнес иелері, дүкен басшылары, маркетологтар, аналитиктер және логистика мамандары.

Сайт бөлімдері: Жүйе бірнеше негізгі блоктардан тұрады. Бақылау панелі (Dashboard) бөлімінде сауданың жалпы көлемі (күнделікті, апталық), «ТОП-5» тауарлар тізімі және қоймадағы қалдықтар көрсетіледі. Клиенттер базасы (CRM) бөлімі сатып алу тарихын, тұрақты клиенттерді анықтауды (Loyalty score) және келу жиілігін бақылауды қамтиды. Сұранысты болжау (Prediction) бөлімінде маусымдық сұраныстар есептеліп, болашақ қажеттіліктер график арқылы көрсетіледі және «тауар тапшылығы» туралы ескертулер беріледі. Маркет-себет талдауы (Basket Analysis) арқылы жиі бірге алынатын тауарлар анықталып, тиімді жеңілдіктер ұсынылады. Аналитика бөлімінде ABC-анализ, Churn Rate (клиенттердің кетуі) және Heatmap (ең белсенді сауда уақыттары) сияқты маңызды көрсеткіштер талданады.

Қолданылатын технологиялар: Интерфейсті жасау үшін HTML, CSS және JavaScript қолданылады. Деректерді өңдеу үшін Python (Pandas, NumPy), ал болжам жасау (Machine Learning) үшін Scikit-learn кітапханасы пайдаланылады. Деректерді сақтау үшін PostgreSQL немесе Firebase технологиялары таңдалған.

Жоба жоспары:  22.05.2026 - 24.05.2026: Деректер жинау, талдау және жобаның негізгі логикасын зерттеу.

25.05.2026 - 28.05.2026: Сайттың интерфейс дизайнын жасау, графиктер мен диаграммаларды сызу.

29.05.2026 - 31.05.2026: Аналитикалық есептерді дайындау, қорытынды есеп пен презентация құрастыру.

01.06.2026 - 02.06.2026: Жобаның функционалын тексеру, қателерді түзету және дайын жұмысты тапсыру.

Күтілетін нәтиже: Бизнеске 15-20% шығынды азайтуға көмектесетін, нақты деректерге сүйенген, интуитивті түсінікті аналитикалық платформа құру.
